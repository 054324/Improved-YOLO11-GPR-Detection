Improved-YOLO11-GPR-Detection (CMDAF-YOLO11)
This repository provides the official implementation of CMDAF-YOLO11, an advanced deep learning framework specifically engineered for Ground Penetrating Radar (GPR) B-scan image detection. By integrating DA-AFPEUA, MFAE modules, and an optimized Loss function, this system significantly enhances the detection of hyperbolic signatures (e.g., UXO, pipelines) within complex subsurface clutter and high-noise environments.
